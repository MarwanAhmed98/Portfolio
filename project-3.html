Smart Login Project

Smart Login by using HTML5 ,CSS3 , Pure JS

Live Link

Project Overview

1. Technologies Used
.HTML5 → Structure for the login form and input fields.
.CSS3 → Styling, animations, and responsiveness.
.Pure JavaScript (Vanilla JS) → Handles form validation, local storage/session storage, and authentication logic.

2. Key Features
1-Modern & Responsive UI
.Uses Flexbox/Grid for a clean and adaptable layout.
.Supports dark mode or theming.
2-Client-Side Form Validation (JavaScript)
.Validates email/password before submission.
.Ensures strong password rules (uppercase, lowercase, numbers, symbols).
3-Local Storage / Session Storage Authentication
.Stores user data securely in localStorage or sessionStorage.
.Maintains login state even after page refresh.
4-Secure Password Handling
.Implements hashing (if needed, using Web Crypto API).
.Prevents XSS (Cross-Site Scripting) attacks.
5-Interactive UI Enhancements
.Password show/hide toggle.
.Animated login button (loading spinner).
.Shake animation for incorrect credentials.

3. Smart Functionalities
.Auto-Remember Me → Saves user credentials securely for faster login.
.Two-Step Verification (Optional) → Email OTP-based login.
.Captcha Integration → Prevents bot logins.
